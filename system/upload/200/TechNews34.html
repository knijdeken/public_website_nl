MemoDial

Memocom also released a special tool for Commence work together with your telephone system for outbound calls. This product is called MemoDial. You (or we can arrange that for you) add buttons to the screen and if you press the button, your telephone will dial a number from the database.

Commence always had already some built in TAPI support but MemoDial goes far beyond that. An overview of the features:

• Automatic processing of telephone numbers (removal of strange characters, 0 and 00 handling);
• Extensive log;
• Support for 3CX, Standard TAPI and AVAYA telephone systems;
• DDE interface back to Commence for workflow;

If your phone is not on the list, please call our support department if you are interested. More information can be found in the latest MemoTools manual.

MemoSendDDE

You can also setup Commence to react on incoming calls. The easiest way to do this is to setup the software of your telephone system to call MemoSendDDE. This program will send a message to Commence and Commence can automatically show for example the information of the caller.

More information can be found in the latest MemoTools manual.
