Special characters in Commence

Commence has always allowed the usage of special characters. Apart from using these characters in datafields, you can even use them in category, field and view names. With the increased number of features in Commence and integration possibilities, more and more problems arise with the usage of these characters. To prevent future issues we urgently advise you to only use the following characters in new category, field and view names:


0-9, a-z, A-Z _ -


We explicitly state new, because you should not start renaming existing category, field and view names, because that might break existing features.

Name field and space

You can add any text in a text field, so it is possible to start with a space. However, it is advised not to do this in a name field, because this is also the (unique) key of the item. So make sure that none of your namefield in categories start with a space, to prevent any problems.
